# Install new Mac

1. Boot up the Mac, adding iCloud.
2. Open App store, install all outstanding updates.
3. Install XCode.
4. Open terminal and paste ```wget https://github.com/fuzzy76/dotfiles/raw/master/scripts/start.sh -O - | bash```
